Html website maken

Deel 1: Frames

Eerst leerden we de basis van html en codeerden we onze eerste website. Al snel leerden we over frames, om zo onze website op te delen en het menu overzichtelijk te maken. We konden dn zelf beginnen met het maken van een menu met verschillende onderdelen. Door het zelf uit te voeren werd het allemaal duidelijk en hadden we al snel een werkende website. Deze website werd dan opgevrolijkt met wat afbeeldingen. De echte lay-out werd dan later gedaan met CSS.

Deel 2: CSS

Nadat we de basis van onze website hadden, was het tijd om de lay-out onder handen te nemen. M.b.v. een CSS file konden we de lay-out loskoppelen van de basis. hierdoor kan je deze gemakkelijk wijzigen zonder de website zelf aan te passen. Dit was vooral handig, wanneer we de titels of achtergrondkleur wouden wijzigen over de volledige website met maar een woord aan te passen.
